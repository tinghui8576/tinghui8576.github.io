Linear Encoding of Facial Expressions: Synthesizing Faces from Neutral to Happy

Project Overview

The project aims to investigate how specific facial features influence human perception and ratings using facial images and a linear encoding model. A subset of 200 images was chosen from FEI Face Database focusing on emotional expressions ranging from neutral to happy. I used the Psychopy package to create a rating system that presented these images to participants for evaluation. After collecting the data, I applied PCA and forward selection to identify key features and reduce dimensionality. A linear regression model was then developed using the selected PCA components to predict ratings, and synthetic face images corresponding to various ratings were generated.

Key Technologies Used
  • Software: Python, Psychopy, PCA, Linear model
