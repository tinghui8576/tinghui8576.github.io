Developing a CNN Model For Apple Leaf Disease Detection

Project Overview

In this course project, I used a dataset from a Kaggle competition to train Convolutional Neural Network models, specifically InceptionV3 and a custom-built model. The goal was to accurately classify each leaf image in the test dataset into its respective disease category and to identify individual diseases when multiple symptoms appear on a single leaf. By comparing the performance of these different models, I try to identify the most effective approach for detecting diseases in apple leaves.

Key Technologies Used
  • Software: Python, Convolutional Neural Network
